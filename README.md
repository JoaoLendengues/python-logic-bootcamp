# 🚀 Python Logic Bootcamp - 4 Semanas

> Um programa estruturado para dominar **lógica de programação em Python** do zero ao intermediário.

## 📊 Sobre Este Programa

Você está aqui porque consegue entender teoria, mas sente dificuldade em **resolver problemas na prática**.

**Este programa vai mudar isso.**

```
SEMANA 1: Condicionais (if/else) + Loops (for)
SEMANA 2: Listas + Loops combinados
SEMANA 3: Dicionários + Listas de dicionários
SEMANA 4: Funções + Integração de tudo
```

---

## ✅ Objetivo Final

Ao final de 4 semanas, você vai conseguir resolver:

```python
# DESAFIO 1 - Filtrar números pares
# [1, 2, 3, 4, 5, 6] → [2, 4, 6]

# DESAFIO 2 - Filtrar produtos por preço
# {'produto1': 50, 'produto2': 150} → ['produto2']

# DESAFIO 3 - Filtrar com múltiplos critérios
# [{'nome': 'X', 'preço': 100}...] → produtos filtrados
```

## 🚀 Como Começar

### Pré-requisitos
- Python 3.8+ instalado
- Git configurado
- Editor de código (VS Code, PyCharm, etc)

### Setup Inicial

```bash
# 1. Clone este repositório
git clone https://github.com/JoaoLendengues/python-logic-bootcamp.git
cd python-logic-bootcamp

# 2. Crie ambiente virtual (RECOMENDADO)
python -m venv venv

# Linux/Mac
source venv/bin/activate

# Windows
venv\Scripts\activate

# 3. Você está pronto! Comece com Semana 1
```

## 📝 Como Fazer Um Exercício

1. **Abra o arquivo do exercício**
   ```bash
   # Exemplo
   code semana_1/exercicios/01_condicional_basico.py
   ```

2. **Leia o problema comentado no arquivo**

3. **Escreva sua solução**

4. **Teste rodando**
   ```bash
   python semana_1/exercicios/01_condicional_basico.py
   ```

5. **Se der erro**, tente de novo (não olhe a solução ainda!)

6. **Se conseguir ou desistir**, veja `SOLUÇÕES.md`

7. **Faça commit**
   ```bash
   git add semana_1/exercicios/01_condicional_basico.py
   git commit -m "feat: semana 1 - exercício 01 completo"
   git push
   ```
